import matplotlib.pyplot as plt
import seaborn as sns
from sklearn.metrics import confusion_matrix, accuracy_score, recall_score

log_reg.fit(X_train, y_train)
y_proba = log_reg.predict_proba(X_test)[:, 1]

def plot_confusion(y_true, y_scores, umbral):
    y_pred = (y_scores >= umbral).astype(int)
    
    cm = confusion_matrix(y_true, y_pred, labels=[0,1])
    
    acc = accuracy_score(y_true, y_pred)
    sens = recall_score(y_true, y_pred, pos_label=1)
    spec = recall_score(y_true, y_pred, pos_label=0)
    
    plt.figure(figsize=(5,5))
    sns.heatmap(cm, annot=True, fmt="d", cmap="Blues", cbar=False,
                xticklabels=["Pred 0","Pred 1"],
                yticklabels=["Real 0","Real 1"])
    plt.title(f"Matriz de confusin (Umbral={umbral})\nAccuracy={acc:.2f}, Sensibility={sens:.2f}, Specificity={spec:.2f}")
    plt.xlabel("Prediccin")
    plt.ylabel("Real")
    plt.show()

for umbral in [0.3, 0.5, 0.7]:
    plot_confusion(y_test, y_proba, umbral)

Los coeficientes de la regresin logstica se interpretan a travs de su valor exponencial (Exp(Coef)), que indica cmo cambia la probabilidad de que ocurra la salida cuando la variable aumenta una unidad:

  • Si Exp(Coef) > 1, la variable aumenta la probabilidad de tener auto.

  • Si Exp(Coef) < 1, la variable disminuye la probabilidad de tener auto.

  • Si Exp(Coef) 1, la variable aporta poco o nada a la prediccin.

En este modelo, por ejemplo, ms cuartos y baos incrementan la probabilidad de tener auto (Exp(Coef) > 1). En contraste, no tener internet o algunas regiones/entidades reducen esa probabilidad (Exp(Coef) < 1). Finalmente, variables como folio o factor_expansion presentan valores cercanos a 1, lo que significa que apenas influyen en el resultado.
